# natsuki.cloud

Personal site built with Astro, TypeScript, and Tailwind CSS.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Cloudflare Pages

Use these settings:

- Framework preset: `Astro`
- Build command: `npm run build`
- Build output directory: `dist`
- Node.js version: current LTS or newer
